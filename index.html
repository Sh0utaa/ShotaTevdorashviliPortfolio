Drivers License Test is a full-stack web application I originally built as my university capstone project — and it evolved into a strong addition to my portfolio. As the name suggests, it simulates the theoretical portion of the Georgian driver’s license exam.

All exam questions are scraped from teoria.on.ge using C#, and the system includes full authentication and authorization, email verification, user statistics, and anti-cheating security measures to prevent users from seeing correct answers or gaming the test.
